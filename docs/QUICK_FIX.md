# 🔧 快速修复：看不到重置密码按钮

## 解决方案

### 方案1：强制刷新浏览器 ⭐推荐
```
Windows: Ctrl + Shift + R 或 Ctrl + F5
Mac: Cmd + Shift + R
```

### 方案2：清除缓存
```
1. 按 F12 打开开发者工具
2. 右键点击浏览器刷新按钮
3. 选择"清空缓存并硬性重新加载"
```

### 方案3：重启开发服务器
```bash
# 停止当前服务器 (Ctrl + C)
# 然后重新启动
cd frontend
npm run dev
```

### 方案4：检查当前代码
打开浏览器开发者工具（F12），查看是否有红色错误。

---

## ✅ 按钮应该在这里

进入管理后台 → 用户管理tab → 往下滚动查看用户卡片

每个用户卡片底部有4个按钮：
```
[ 禁用账户/启用账户 ]  [ 🔑 重置密码 ]
[ 查看详情 ]           [ 🗑️ 删除 ]  (仅超级管理员可见)
```
