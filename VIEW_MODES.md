# 📋 文件视图模式说明

## 支持的视图模式

### 1. 🎨 网格视图 (Grid View) - 默认
**适用场景：**
- 浏览图片文件
- 需要预览缩略图
- 视觉化浏览

**特点：**
- ✅ 大尺寸缩略图预览
- ✅ 16:10 卡片比例
- ✅ 悬停显示快速操作
- ✅ 响应式网格布局
- ✅ 适合图片和视觉内容

**布局：**
- 桌面：4-5 列
- 平板：3 列
- 手机：2 列

---

### 2. 📄 列表视图 (List View)
**适用场景：**
- 查看文件详细信息
- 需要快速扫描文件名
- 管理大量文件

**特点：**
- ✅ 横向排列
- ✅ 显示完整文件名
- ✅ 文件大小和日期
- ✅ 快速操作按钮
- ✅ 适合文档管理

**信息显示：**
- 缩略图/图标
- 文件名
- 文件大小
- 创建日期
- 操作按钮（分享、下载、删除）

---

### 3. 📊 紧凑视图 (Compact View)
**适用场景：**
- 浏览大量文件
- 需要一屏显示更多内容
- 快速查找文件

**特点：**
- ✅ 最小化空间占用
- ✅ 一行显示一个文件
- ✅ 只显示关键信息
- ✅ 快速操作
- ✅ 适合文件列表

**信息显示：**
- 文件图标
- 文件名
- 文件大小
- 操作按钮（下载、删除）

---

## 视图切换

### 桌面端
- 点击工具栏右上角的视图切换按钮
- 三个图标分别代表：网格、列表、紧凑

### 移动端
- 同样在工具栏显示
- 自动适配触摸操作
- 记住用户选择

### 快捷键（计划中）
- `Ctrl/Cmd + 1` - 网格视图
- `Ctrl/Cmd + 2` - 列表视图
- `Ctrl/Cmd + 3` - 紧凑视图

## 自动保存

用户选择的视图模式会自动保存到浏览器本地存储（localStorage），下次访问时自动恢复。

```javascript
// 保存视图模式
localStorage.setItem('fileViewMode', 'grid')

// 读取视图模式
const mode = localStorage.getItem('fileViewMode') || 'grid'
```

## 响应式适配

### 桌面端 (>768px)
- 所有视图模式完整显示
- 网格视图：4-5 列
- 列表视图：完整信息
- 紧凑视图：单行显示

### 移动端 (<768px)
- 网格视图：2 列
- 列表视图：垂直堆叠信息
- 紧凑视图：简化操作按钮

## 使用建议

### 图片管理
推荐：**网格视图**
- 大缩略图便于识别
- 视觉化浏览体验
- 快速找到目标图片

### 文档管理
推荐：**列表视图** 或 **紧凑视图**
- 完整文件名显示
- 快速扫描文件列表
- 高效的空间利用

### 混合内容
推荐：**列表视图**
- 平衡视觉和信息
- 适合各种文件类型
- 操作便捷

## 技术实现

### 组件结构
```
FileList.vue
├── 视图工具栏
│   ├── 视图切换按钮
│   └── 文件计数
├── 网格视图
│   └── 文件卡片
├── 列表视图
│   └── 文件行
└── 紧凑视图
    └── 紧凑行
```

### 状态管理
- 使用 Vue 3 Composition API
- 响应式视图切换
- localStorage 持久化

### 性能优化
- 虚拟滚动（大列表）
- 图片懒加载
- 防抖处理
- 条件渲染
