## 🚀 如何开始

### 1. 快速启动（5 分钟）

```bash
# 1. 设置 Supabase（参考 QUICKSTART.md）
# 2. 配置环境变量
cd frontend
cp .env.example .env
# 编辑 .env 填入 Supabase 配置

# 3. 安装依赖
npm install

# 4. 启动开发服务器
npm run dev
```
